установить гит

1. сначала создаем репу 
2. git cd (поменять директорию)  
3. git clone создаем клона  (в папке должен быть readme файл)
4. добавляем наши файлы и папки
5. git status (проверка)
6. git add . (доавление новых файлов и подготовка их)
7. git commit -am "nazvanie" (сохранение изменений)
8. git push (отправить файлы)
